# ChurchOS

> Church management and media presentation — unified.

**ChurchOS** is a hybrid platform combining the media power of dedicated presentation software (like EasyWorship and Pewbeam) with modern church administration tools, all in one place.

Built by [Aisida Studios](https://aisidastudios.uk).

---

## The Problem

Churches typically juggle two separate tools:

- A **presentation app** (EasyWorship, ProPresenter, Pewbeam) for song lyrics, Scripture, and slides during service
- A **church management system** for rotas, attendance, announcements, and communications

ChurchOS brings both into a single platform — so your media team and admin team are working from the same place.

---

## Planned Features

### Media & Presentation
- Song lyrics display with live control
- Scripture lookup and display
- Custom slide builder for announcements and notices
- Multi-screen output support

### Church Management
- Rota and team scheduling
- Attendance tracking
- Announcements and notice board
- Member directory

### Platform
- Web-based (no installs required)
- Role-based access (admin, media team, general member)
- Mobile-friendly control panel for live service management
